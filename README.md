# Animove 2025

In this repository you can find the lectures (presentations and R scripts) of AniMove 2025.

To easily install all libraries needed to run these scripts, you can install the release "Animove2025" of the animove_R_package https://github.com/AniMoveCourse/animove_R_package/releases/tag/Animove2025

The lectures of AniMove 2024 were recorded and are available under `eLearning` on https://animove.org/
